Проект содержит 10 тестов для сайта otus.ru.

Проект содержит тесты с использованием следующих компонентов:
- Spring: для создания экземпляров классов страниц и драйвера
- Cucumber: для создания сценариев тестов с BDD подходом
- PageObject: для обращения к элементам страницы

Работа с драйвером вынесена в отдельный класс utils/Driver.
В steps/BasePageStepDefs вынесены хуки по закрытию драйвера, также используется аннотация
@DirtiesContext для очистки контекста (чтобы пересоздавался bean драйвера).
В resources/features находятся BDD сценарии тестов.
Шаги в сценариях максимально переиспользуются.
